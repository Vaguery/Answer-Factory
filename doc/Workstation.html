Workstation

Public Class Methods

new (name: Symbol) {|w| config } → workstation click to toggle source

Creates a new workstation, and stores it in Factory::Workstations with the key name.

If a config block is provided, it runs at the end of initialization, passing the newly created workstation as its parameter. 

  w = Workstation.new(:w) do |workstation|
    Machine.new(:m, w) do |m|
      ...
    end
    
    w.schedule :m
  end
  
  w.inspect       #=> #<Workstation:0x10012bdc0 @name=:w,
                        @machines={:m1=>#<Machine:0x100490128>}
                        @schedule=[:m]>
