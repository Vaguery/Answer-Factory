score (name: Symbol or {name: Symbol => value, * }) → value or {name: Symbol => value, * } click to toggle source

Given a hash argument, associates the score names with new Score instances containing the associated values, then returns the hash. 

  answer = Answer.new("ref x", "nudge")
  
  answer.score(:x => 5)   #=> {:x=>5}

Given a symbol argument, returns the value of the named score in answer’s scores hash. 

  answer.score(:x)        #=> 5.0

If no answer exists with the given name, returns Factory::Infinity. 

  answer.score(:y)        #=> 3.0e+38
